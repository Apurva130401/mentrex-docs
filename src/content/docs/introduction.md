# Introduction to Mentrex

## What is Mentrex?
Mentrex is an advanced autonomous coding agent designed to live inside your IDE. It acts as a pair programmer that can not only generate code but also:
- **Execute Commands**: Build, test, and deploy applications.
- **Manage Files**: Create, edit, and explore your codebase.
- **Browse the Web**: Research documentation and solutions.

## Core Philosophy
We believe developers should spend less time on boilerplate and configuration and more time on creative problem solving. Mentrex bridges the gap between AI and your local development environment.

## Key Features
- **Context Awareness**: Understands your entire project structure.
- **Secure**: Uses API keys and local execution, keeping you in control.
- **Flexible**: Compatible with VS Code and other modern editors.
